You need to install adafruit's gps library:
	- sudo pip install pip install adafruit-circuitpython-gps

Other shell commands to run:
	- sudo apt-get install gpsd gpsd-clients python-gps
	- sudo systemctl stop gpsd.socket
	- sudo systemctl disable gpsd.socket
	- sudo gpsd /dev/ttyUSB0 -F /var/run/gpsd.sock

Now we will test whether the previous commands worked:
	- cgps -s
Now you should see a box with your gps position and parameters 
(give it a sec). If the parameters are shown you have installed it 
correctly.... 

This command has to be run before using the gps library:
	- sudo gpsd /dev/ttyUSB0 -F /var/run/gpsd.sock


For further infomation visit:
	- https://learn.adafruit.com/adafruit-ultimate-gps-on-the-raspberry-pi?view=all
